# Echowhy V1.5 全局悬浮提问窗 PRD（含双主题适配）

## 版本：1.0

## 核心原则

**完全保留 "常驻左下角 + 拖动插入 + 提示线" 的核心交互逻辑**，不新增任何多余元素，不破坏极简深空风格。仅优化交互丝滑度、精准度和双主题一致性，确保白天 / 黑夜模式下体验完全统一。

------

## 一、基础规范（双主题通用）

### 1. 核心定位

这是产品**唯一的提问入口**，没有任何系统预设的提问位置。用户的疑问在哪里，就可以将问题钉在哪里，完全由用户的思考节奏主导。

### 2. 固定位置

- 永久悬浮在**中间主内容区的左下角**
- 距离内容区底部：20px
- 距离内容区左侧：20px
- 永远不会遮挡正文文字（正文自带左右内边距）

### 3. 基础结构

plaintext











```
[全局悬浮提问窗]
├─ 顶部拖动栏：MY QUESTION 标题
├─ 输入框：提示文字 Drop one sharp why here...
└─ 底部按钮：[ CANCEL ] + [ INSERT ]
```

------

## 二、双主题视觉规范

### 1. 黑夜主题（默认）

- 背景：半透毛玻璃 `bg-black/60 backdrop-blur-md`
- 边框：1px 实线 `border-cyan-900/50`
- 标题文字：蓝色大写 `text-cyan-400`
- 输入框文字：白色 `text-white`
- 提示文字：灰色 `text-gray-400`
- 按钮样式：与通用问题卡片的 `[ CHECK ]` 按钮完全一致
- 插入提示线：1px 实线 `#60a5fa`（与左侧星座链颜色一致）

### 2. 白天主题

- 背景：半透毛玻璃 `bg-white/80 backdrop-blur-md`
- 边框：1px 实线 `border-blue-200`
- 标题文字：蓝色大写 `text-blue-600`
- 输入框文字：黑色 `text-gray-900`
- 提示文字：灰色 `text-gray-500`
- 按钮样式：与通用问题卡片的 `[ CHECK ]` 按钮完全一致
- 插入提示线：1px 实线 `#3b82f6`（与白天主题星座链颜色一致）

------

## 三、交互优化规范（双主题通用）

### 1. 常驻状态优化：智能半隐

- 非交互状态：保持 `opacity-40` 半透明，不干扰阅读
- 鼠标悬停状态：平滑过渡到 `opacity-100` 完全显示
- 过渡动画时长：200ms，自然流畅

### 2. 拖动交互优化

- **可拖动区域**：仅顶部标题栏（"MY QUESTION" 区域）可拖动，点击输入框仅激活光标，不触发拖动

- **拖动延迟**：按住标题栏后延迟 100ms 再触发拖动，防止误触

- **拖动态样式**：拖动过程中整体缩小 15%，减少视线遮挡；z-index 提升至最高，不被任何元素遮挡

- 提示线反馈

  ：

  - 拖动到可插入位置时，提示线自动变粗 1px、亮度提高 30%
  - 自动吸附到最近的行间隙 / 段落间隙，无需用户精准对齐

  

- 双模式插入

  ：

  - 拖动到**两个段落之间**：显示全宽横向提示线，插入到段落之间
  - 拖动到**某一行文字下方**：显示与该行等宽的短提示线，插入到该行正下方

  

### 3. 取消操作优化

- 按下 `ESC` 键：立即取消插入，悬浮窗平滑回到左下角原位
- 拖动到**中间内容区以外**（左侧导航 / 右侧源码区）：松开鼠标自动取消
- 拖动过程中点击鼠标右键：立即取消插入
- 点击 `[ CANCEL ]` 按钮：清空输入框，悬浮窗回到原位

### 4. 插入动画与状态一致性

- 点击 `[ INSERT ]` 后，悬浮窗**原地平滑过渡为标准的 `● MY QUESTION` 通用待回答卡片**
- 同时，一个新的空白悬浮窗从左下角淡入出现，准备下一次提问
- 所有样式、字体、按钮与通用问题卡片 100% 统一，无视觉割裂

------

## 四、绝对禁止项

1. 禁止添加任何其他提问入口（如节点末尾的 `[ + WHY ]` 按钮）
2. 禁止修改悬浮窗的核心结构和拖动逻辑
3. 禁止在任何主题下使用与产品颜色体系不一致的颜色
4. 禁止添加任何多余的动画或视觉效果
5. 禁止改变悬浮窗的固定位置

------

## 五、验收标准

### 通用验收

1. 悬浮窗永久固定在中间内容区左下角，不遮挡正文
2. 仅顶部标题栏可拖动，点击输入框不触发拖动
3. 支持在任意段落、任意行下方插入问题，提示线精准吸附
4. 提供 ESC、拖出区域、右键、CANCEL 四种取消方式
5. 插入后原地生成标准的 MY QUESTION 卡片，同时新悬浮窗自动出现

### 黑夜主题验收

1. 背景为半透黑色毛玻璃，边框为淡蓝色
2. 插入提示线为 `#60a5fa`，与星座链颜色一致
3. 文字颜色符合规范，清晰可读

### 白天主题验收

1. 背景为半透白色毛玻璃，边框为淡蓝色
2. 插入提示线为 `#3b82f6`，与星座链颜色一致
3. 文字颜色符合规范，清晰可读
