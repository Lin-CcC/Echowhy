# Echowhy V1.5 Library 学习库卡片终极优化 PRD（双主题・星空全息版）

## 版本：1.0

## 核心原则

**吸收双方方案所有优点，保留全部刚需功能，严格遵循星空全息玻璃质感，双主题 100% 适配，不新增任何非 MVP 功能**。所有改动仅通过 Tailwind 类名实现，不改变现有网格布局和数据库结构。

------

## 一、核心共识与取舍

### ✅ 必须保留的优点

1. **全息玻璃质感**：替代死板的纯黑背景，注入星空通透感
2. **底部进度条**：神来之笔，用最简洁的方式直观展示完成度
3. **右上角数字进度**：不占用内部空间，信息密度最高
4. **标签化源码**：比纯文本更清晰，视觉层次更好
5. **悬浮外发光效果**：完美契合深空主题，提升高级感
6. **双按钮功能区分**：`Open`（继续学习）和`Ask New`（新建提问）各司其职，绝不混淆
7. **完整状态体系**：明确区分未开始 / 进行中 / 已完成
8. **基础管理功能**：右键菜单、排序、搜索

### ❌ 必须砍掉的非刚需 / 有问题设计

1. **AI 自动摘要**：超出 MVP 范围，实现成本高，非核心需求
2. **默认隐藏的动态按钮**：降低可发现性，用户不知道可以点击
3. **环形进度条**：实现复杂，不如底部进度条简洁直观
4. **右上角更多按钮**：用右键菜单替代，保持界面干净

------

## 二、卡片信息结构规范（终极版）

plaintext











```
[学习模块卡片]
├─ 右上角：数字进度（如 3/7 | 42%）
├─ 第一行：模块标题（用户自定义） + 已完成标识 ✓
├─ 第二行：源码标签（微型胶囊样式）
├─ 第三行：最后学习时间（相对时间）
└─ 最底部：全宽进度条 + 底部按钮区 [ Open ] [ Ask New ]
```

### 各层级样式优先级

表格







| 元素         | 黑夜主题                        | 白天主题                        | 字号    | 字重 |
| :----------- | :------------------------------ | :------------------------------ | :------ | :--- |
| 右上角进度   | `text-cyan-400`                 | `text-cyan-600`                 | text-sm | 500  |
| 模块标题     | `text-white`                    | `text-gray-900`                 | text-lg | 600  |
| 源码标签     | `bg-slate-800/50 text-gray-300` | `bg-slate-200/50 text-gray-600` | text-xs | 400  |
| 最后学习时间 | `text-gray-500`                 | `text-gray-500`                 | text-xs | 400  |
| 已完成标识 ✓ | `text-cyan-400`                 | `text-cyan-600`                 | text-sm | 400  |

## 三、视觉规范（星空全息质感・双主题）

### 1. 基础卡片样式（所有状态通用）

css











```
/* 黑夜主题 */
.card-base {
  @apply bg-slate-900/40 backdrop-blur-md border border-slate-700/50 rounded-md;
  @apply transition-all duration-300 ease-out;
}

/* 白天主题 */
.card-base-light {
  @apply bg-white/70 backdrop-blur-md border border-slate-200/70 rounded-md;
  @apply transition-all duration-300 ease-out;
}
```

### 2. 状态区分规范（边框 + 进度条颜色）

表格







| 状态   | 黑夜主题                                          | 白天主题                                          |
| :----- | :------------------------------------------------ | :------------------------------------------------ |
| 未开始 | 边框：`border-slate-700/50`进度条：`bg-slate-600` | 边框：`border-slate-300/50`进度条：`bg-slate-400` |
| 进行中 | 边框：`border-blue-500/60`进度条：`bg-blue-500`   | 边框：`border-blue-600/60`进度条：`bg-blue-600`   |
| 已完成 | 边框：`border-cyan-500/60`进度条：`bg-cyan-500`   | 边框：`border-cyan-600/60`进度条：`bg-cyan-600`   |

### 3. 底部进度条规范

- 位置：卡片最底部，绝对定位，全宽
- 高度：`h-1`
- 宽度：`width: ${progress}%`，动态计算
- 圆角：`rounded-b-md`（与卡片底部圆角对齐）
- 已完成模块：进度条宽度 100%

### 4. 悬浮交互规范（核心高级感来源）

css











```
/* 黑夜主题悬浮 */
.card-base:hover {
  @apply -translate-y-1 border-cyan-500/40;
  @apply shadow-[0_8px_30px_rgba(34,211,238,0.1)];
}

/* 白天主题悬浮 */
.card-base-light:hover {
  @apply -translate-y-1 border-cyan-600/40;
  @apply shadow-[0_8px_30px_rgba(34,211,238,0.15)];
}
```

### 5. 按钮规范

- 位置：卡片底部右侧，并排显示

- 间距：两个按钮间距 8px

- 样式：

  - `[ Open ]`：次按钮，灰色边框，与主界面`[ SKIP FOR NOW ]`完全一致
  - `[ Ask New ]`：主按钮，蓝色边框，与主界面`[ CHECK ]`完全一致

  

- 悬浮：按钮边框颜色同步提亮，与卡片悬浮效果呼应

## 四、交互规范

### 1. 点击交互

- 点击卡片任意空白区域：默认执行`Open`功能
- 点击`[ Open ]`：打开完整学习界面，恢复所有历史进度、回答记录、星座链状态
- 点击`[ Ask New ]`：直接跳转到提问界面，自动加载该模块源码，清空历史进度

### 2. 右键菜单

- 右键点击卡片任意区域，弹出极简右键菜单（与主界面样式完全一致）

- 菜单选项：

  1. `Rename`：修改模块标题
  2. `Delete`：带确认提示永久删除
  3. `Open in new tab`：新标签页打开

  

### 3. 全局功能

- 页面顶部排序下拉菜单：最近创建 / 最近打开 / 学习进度
- 页面顶部搜索框：支持按模块标题和源码名称搜索
- 日期格式：统一使用相对时间（Just now / X hours ago / X days ago）

### 4. 空状态

plaintext











```
No learning modules yet
Go to Start page to import your first source code
```

下方添加`[ GO TO START ]`主按钮，点击跳转到 Start 页面

## 五、绝对禁止项（严格执行）

1. 禁止保留右上角的`SOURCE`静态标签
2. 禁止添加任何 AI 自动摘要功能
3. 禁止使用默认隐藏的动态按钮
4. 禁止使用环形进度条
5. 禁止添加任何多余的装饰元素和图标（除已完成的`✓`）
6. 禁止改变现有卡片网格布局和间距
7. 禁止使用产品现有色板以外的任何颜色

## 六、验收标准

### 通用验收

1. 卡片信息层级清晰，扫一眼即可获取标题、源码、进度、状态
2. 三种状态（未开始 / 进行中 / 已完成）视觉区分明确
3. 卡片底部有全宽进度条，右上角有数字进度
4. 源码显示为微型胶囊标签样式
5. 卡片底部有`[ Open ]`和`[ Ask New ]`两个按钮
6. 点击任意区域、两个按钮功能正常
7. 右键菜单支持重命名、删除、新标签页打开
8. 排序和搜索功能正常
9. 日期显示为相对时间
10. 空状态显示正常

### 视觉验收

1. 卡片使用全息玻璃毛玻璃背景，不是纯实心
2. 悬浮时卡片上浮 1px，边框变亮，出现青色外发光阴影
3. 底部进度条与卡片底部圆角对齐
4. 所有文字颜色符合规范，清晰可读
5. 整体视觉风格与主界面星空主题完全统一

### 双主题验收

1. 切换主题时，所有颜色自动切换
2. 白天主题背景为半透白色，黑夜主题为半透深灰
3. 进度条、边框、文字颜色在两个主题下都符合规范
4. 悬浮效果在两个主题下都正常显示

### 功能验收

1. `Open`正确恢复所有历史学习状态
2. `Ask New`正确跳转到提问界面并加载源码
3. 重命名和删除功能正常（删除带确认提示）
4. 排序功能按要求正确排序
5. 搜索功能实时过滤卡片
